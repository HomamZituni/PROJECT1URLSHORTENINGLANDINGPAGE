Description
URL Shortener: A landing page with bitly API integration that allows user to shorten long URLs and list them out on the page with localstorage persistence.

## Features
Users can enter their long URL and with the click of a button create and copy a shortened version.
Users can also create lists of previously created shortened URLs and delete them if needed.
